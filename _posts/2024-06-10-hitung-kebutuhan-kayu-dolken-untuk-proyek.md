Salah satu pertanyaan paling sering dari customer adalah: "Berapa banyak kayu dolken yang saya butuhkan?" Artikel ini akan memberikan panduan lengkap untuk menghitung kebutuhan material dengan akurat.

## Mengapa Perhitungan Akurat Penting?

### Jika Terlalu Sedikit:

- ❌ Project terhenti di tengah jalan
- ❌ Harus order lagi (ongkir tambahan)
- ❌ Risiko perbedaan batch/warna
- ❌ Delayed timeline

### Jika Terlalu Banyak:

- ❌ Pemborosan budget
- ❌ Storage problem
- ❌ Material terbuang (jika tidak ada project lain)

### Perhitungan Tepat:

- ✅ Budget akurat
- ✅ No delays
- ✅ Efisiensi maksimal
- ✅ Profesional execution

## Formula Dasar

### Konsep Penting:

1. **Jumlah Batang** = Total Panjang Needed ÷ Panjang per Batang
2. **Buffer** = 10-15% dari total (untuk cutting loss, kerusakan, dll)
3. **Final Order** = Jumlah Batang + Buffer

**Standar Panjang**: 4 meter per batang

## Perhitungan untuk Berbagai Aplikasi

### 1. Pagar Vertikal

**Scenario**: Pagar taman setinggi 1.5 meter, panjang 10 meter

#### Step-by-Step:

**Step 1: Hitung jumlah dolken per meter**

```
Diameter dolken = 6 cm = 0.06 meter
Spacing yang diinginkan = 5 cm = 0.05 meter
Pitch (dolken + spacing) = 0.06 + 0.05 = 0.11 meter

Jumlah per meter = 1 ÷ 0.11 = 9.09 ≈ 9 batang per meter
```

**Step 2: Total untuk panjang pagar**

```
Total per row = 9 batang × 10 meter = 90 batang
```

**Step 3: Adjust untuk tinggi**

```
Tinggi pagar = 1.5 meter
Panjang dolken = 4 meter
1 batang bisa untuk 4 ÷ 1.5 = 2.67 ≈ 2 pieces (dengan sisa)

Actual kebutuhan = 90 ÷ 2 = 45 batang utuh
```

**Step 4: Tambahkan buffer 15%**

```
Buffer = 45 × 15% = 6.75 ≈ 7 batang
Total order = 45 + 7 = 52 batang
```

**Final: Order 52 batang dolken 6-8 cm**

**Budget**: 52 × Rp 30.000 = **Rp 1.560.000**

### 2. Feature Wall Dekoratif

**Scenario**: Wall cladding 3m × 4m = 12 m²

#### Vertical Installation:

**Step 1: Hitung coverage per batang**

```
Diameter = 3 cm = 0.03 m
Panjang = 4 m
Coverage per batang = 0.03 × 4 = 0.12 m²
```

**Step 2: Total batang needed**

```
Luas wall = 12 m²
Batang needed = 12 ÷ 0.12 = 100 batang
```

**Step 3: Adjust untuk spacing (jika ada)**

Jika pakai spacing 1 cm antara dolken:

```
Effective coverage = (0.03 + 0.01) × 4 = 0.16 m²
Batang needed = 12 ÷ 0.16 = 75 batang
```

**Step 4: Buffer**

```
Buffer 10% = 75 × 10% = 7.5 ≈ 8 batang
Total = 75 + 8 = 83 batang
```

**Final: Order 85 batang dolken 2-3 cm** (rounded up)

**Budget**: 85 × Rp 15.000 = **Rp 1.275.000**

### 3. Gazebo

**Scenario**: Gazebo 3m × 3m, tinggi 3m

#### Breakdown Components:

**A. Tiang Utama (8-10 cm)**

```
4 tiang utama @ 3 meter tinggi
Kebutuhan = 4 × 3m = 12 meter linear
Batang needed = 12 ÷ 4 = 3 batang

Order: 4 batang (include 1 buffer)
Cost: 4 × Rp 35.000 = Rp 140.000
```

**B. Rangka Atap (6-8 cm)**

```
Perimeter = (3 + 3 + 3 + 3) = 12 meter
Cross beams = 2 × 3 meter = 6 meter
Diagonal supports = 2 × 4.24 meter = 8.48 meter

Total = 12 + 6 + 8.48 = 26.48 meter
Batang = 26.48 ÷ 4 = 6.62 ≈ 7 batang
Buffer 15% = 7 × 1.15 = 8.05 ≈ 9 batang

Order: 9 batang
Cost: 9 × Rp 30.000 = Rp 270.000
```

**C. Dekorasi Railing (4-6 cm)**

```
Perimeter = 12 meter
Vertical slats @ 10cm spacing = 120 slats
Height per slat = 1 meter

Total linear = 120 × 1 = 120 meter
Batang = 120 ÷ 4 = 30 batang
Buffer 10% = 30 × 1.1 = 33 batang

Order: 33 batang
Cost: 33 × Rp 25.000 = Rp 825.000
```

**Total Gazebo**:
- Material: Rp 1.235.000
- Batang total: 46 batang (mixed sizes)

---

### 4. Pergola

**Scenario**: Pergola 4m × 3m

#### Components:

**Tiang (10-12 cm)**:

```
4 tiang @ 3m = 12 meter linear
Batang = 12 ÷ 4 = 3 batang
Order: 4 batang (include buffer)
Cost: 4 × Rp 45.000 = Rp 180.000
```

**Beam Utama (8-10 cm)**:

```
2 beams @ 4m = 8 meter
Batang = 8 ÷ 4 = 2 batang
Order: 3 batang (include buffer)
Cost: 3 × Rp 35.000 = Rp 105.000
```

**Rafters (6-8 cm)**:

```
Spacing 40cm = 4 ÷ 0.4 = 10 rafters
@ 3m each = 30 meter linear
Batang = 30 ÷ 4 = 7.5 ≈ 8 batang
Order: 9 batang (with buffer)
Cost: 9 × Rp 30.000 = Rp 270.000
```

**Total Pergola**:
- Material: Rp 555.000
- Batang total: 16 batang

---

### 5. Ceiling Treatment

**Scenario**: Cafe ceiling 8m × 6m = 48 m², parallel pattern

**Step 1: Determine spacing**

```
Spacing between dolken = 20 cm
```

**Step 2: Calculate rows needed**

```
Width = 6 meter = 600 cm
Rows = 600 ÷ 20 = 30 rows
```

**Step 3: Length per row**

```
Length = 8 meter
```

**Step 4: Total batang**

```
Total linear = 30 rows × 8m = 240 meter
Batang (4m each) = 240 ÷ 4 = 60 batang
Buffer 10% = 60 × 1.1 = 66 batang
```

**Final: Order 66 batang dolken 3-4 cm**

**Budget**: 66 × Rp 25.000 = **Rp 1.650.000**

## Calculator Tool (Simplified)

### Formula Reference:

#### Pagar Horizontal/Vertikal:

```
Batang = (Panjang × Tinggi) ÷ (Diameter + Spacing) ÷ Panjang_Batang × Buffer
```

#### Wall Cladding:

```
Batang = Luas_Wall ÷ (Diameter × Panjang_Batang) × Buffer
```

#### Ceiling:

```
Batang = (Total_Linear_Length ÷ Panjang_Batang) × Buffer
```

#### Struktur (Gazebo/Pergola):

```
Breakdown per komponen, sum total
```

## Tips Menghitung

### 1. Measure Dua Kali

- Ukur area/panjang multiple times
- Cross-check measurements
- Account untuk irregular shapes

### 2. Gambar Sketch

- Visual reference membantu
- Easier untuk spot errors
- Useful untuk komunikasi dengan supplier

### 3. Break Down Complex Projects

- Jangan hitung sekaligus
- Per section atau component
- Easier to verify

### 4. Konsultasi dengan Supplier


## Common Mistakes

### Mistake 1: Lupa Hitung Cutting Loss

**Issue**: Assume 100% material usable

**Reality**: Selalu ada waste dari cutting

**Solution**: Add 10-15% buffer

### Mistake 2: Tidak Pertimbangkan Spacing

**Issue**: Hitung rapat-rapat tanpa gap

**Reality**: Design biasanya butuh spacing untuk estetika/sirkulasi

**Solution**: Include spacing in calculation

### Mistake 3: Salah Unit

**Issue**: Mix meter, cm, inch

**Reality**: Chaos!

**Solution**: Standardize (recommend: meter untuk all)

### Mistake 4: Tidak Pertimbangkan Struktur Pendukung

**Issue**: Hanya hitung material utama

**Reality**: Butuh crossbeams, supports, dll

**Solution**: List all components
