1 × H2: Main section title (header card) - 1 × H3: Overview title (optional) - 2 × H3: Card titles (persiapan, instalasi) - 8 × H4: Detail headings (4 per card: detail_1 to detail_4) - 0 × H5, H6: Not used in this block Total Headings: 12 (with overview) or 11 (without overview) Paragraphs (REQUIRED for SEO): - 1 intro paragraph (optional) - 4 card paragraphs (paragraph_1, paragraph_2 × 2 cards) - 8 detail paragraphs (detail_1.paragraph to detail_4.paragraph × 2 cards) Total Paragraphs: 13 (with all optional fields) Features: --------- - Gradient header with icon, title (H2), and subtitle - H3 overview section (optional) - 2 hardcoded cards (persiapan, instalasi) - Layout: 2-column (col-lg-6) side-by-side responsive grid - Each card has: - H3 title with colored header - 2 paragraphs for context - 4 detail sections (H4 + paragraph) - Dynamic gradient colors - Icon-based visual design - Hover lift effect Parameters: ----------- - page.cara_pemasangan: Object containing: - title: H2 main section title → class="title" - subtitle: Subtitle below title (optional) → class="subtitle" - intro: Introduction paragraph → class="intro" - icon: Bootstrap icon class - gradient: Color theme (primary, info, warning, success, danger, wood) - overview_title: H3 overview title (optional) → class="overview_title" - overview_subtitle: Overview subtitle (optional) - overview_icon: Overview icon (optional) - overview_color: Overview color (optional) - persiapan: Card 1 (Persiapan) → class="persiapan" - title: H3 card title - icon: Bootstrap icon class - color: Card color - paragraph_1: Text content → class="paragraph_1" - paragraph_2: Text content → class="paragraph_2" - detail_1, detail_2, detail_3, detail_4: Details → class="detail_1", etc - h4: Detail heading → class="h4" - paragraph: Detail text → class="paragraph" - instalasi: Card 2 (Instalasi) → class="instalasi" -
