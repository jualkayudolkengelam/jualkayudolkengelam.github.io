{% comment %} ============================================================================ Mengenal Ukuran Block Component ============================================================================ @file _includes/components/tutorial/block--mengenal-ukuran.html @description Menyajikan informasi lengkap tentang berbagai ukuran produk dengan 5 kartu. Membantu pengguna memahami karakteristik, kegunaan, dan rekomendasi untuk setiap ukuran. Ideal untuk artikel panduan pemilihan ukuran dan katalog produk. @version 1.0.0 @date 2025-11-21 @author jualkayudolkengelam Heading Structure (SEO & Accessibility): ----------------------------------------- - 1 × H2: Main section title (header card) - 1 × H3: Overview title (optional) - 5 × H3: Card titles (card_1 to card_5) - 15 × H4: Detail headings (3 per card: detail_1, detail_2, detail_3) - 0 × H5, H6: Not used in this block Total Headings: 22 (with overview) or 21 (without overview) Paragraphs (REQUIRED for SEO): - 1 intro paragraph (optional) - 10 card paragraphs (paragraph_1, paragraph_2 × 5 cards) - 15 detail paragraphs (detail_1.paragraph to detail_3.paragraph × 5 cards) Total Paragraphs: 26 (with all optional fields) Features: --------- - Gradient header with icon, title (H2), and subtitle - H3 overview section (optional) - 5 hardcoded cards (card_1, card_2, card_3, card_4, card_5) - Layout: 4 cards (col-lg-6) + 1 card (col-lg-12 full width) - Each card has: - H3 title with colored header - 2 paragraphs for context - 3 detail sections (H4 + paragraph) - Dynamic gradient colors - Icon-based visual design - Hover lift effect - Responsive card grid layout Parameters: ----------- - page.mengenal_ukuran: Object containing: - title: H2 main section title → class="title" - subtitle: Subtitle below title (optional) → class="subtitle" - intro: Introduction paragraph → class="intro" - icon: Bootstrap icon class - gradient: Color theme (primary, info, warning, success, danger, wood) - overview_title: H3 overview title (optional) → class="overview_title" - overview_subtitle: Overview subtitle (optional) - overview_icon: Overview icon (optional) - overview_color: Overview color (optional) - card_1, card_2, card_3, card_4, card_5: Hardcoded cards (NO LOOP!) → class="card_1", etc Each card: - title: H3 card title - icon: Bootstrap icon class - color: Card color - paragraph_1: Text content → class="paragraph_1" - paragraph_2: Text content → class="paragraph_2" - detail_1, detail_2, detail_3: Details → class="detail_1", "detail_2", "detail_3" - h4: Detail heading → class="h4" - paragraph: Detail text → class="paragraph" Class Mapping (Best Practice Pattern): --------------------------------------- Frontmatter key → HTML class name - title →
